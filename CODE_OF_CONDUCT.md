# Code of Conduct

RoadForge contributors must communicate professionally and constructively.

Unacceptable behavior includes harassment, discrimination, threats, doxxing,
deliberate disruption, and publishing another person's private information or
credentials.

Maintainers may edit, remove, or reject contributions and restrict participation when
behavior harms the project or its community. Report conduct concerns privately through
the repository's GitHub Security Advisory/private reporting channel. Do not include
unrelated vulnerability details in a conduct report.

This policy applies in repository issues, pull requests, discussions, and other
project-managed spaces.
